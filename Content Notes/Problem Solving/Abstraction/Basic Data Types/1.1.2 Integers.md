## 1.1.2 Basic Data Types: Integers

- Assignment: ` integer = 10 `

- Operations:

  - Basically the same as numbers in real life. However, do take note that in Python 2, the `/` symbol will only return the quotient of the division. In Python 3, the `\` symbol will return the value of the division in a float. For example:

    Python 2: 5 / 2 = 2

    Python 3: 5 / 2 = 2.5
